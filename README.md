# 거제교육지원청 영재교육원 중등수학 2 — 수업 자료

거제교육지원청 영재교육원 **중등수학 2** 수업 자료를 모은 리포지토리입니다.
각 수업은 4차시(180분) 블록, **WEDCR**(Warm-up → Explore → Discover → Connect → Reflect)
교육과정 틀로 설계된 인터랙티브 학습 자료입니다.

- **대상**: 중학교 2학년 영재교육원 학생
- **담당 강사**: 이동언

---

## 📚 수업 목록

| 주차 | 주제 | 내용 | 폴더 |
|------|------|------|------|
| 1주차 | 우리들의 암호 이야기 | 암호학 — 체크섬·시저·빈도분석·RSA | [`crypto/`](crypto/) |
| 2주차 | 기본 다각형과 틱택토 | 위상수학 — 원기둥·토러스·뫼비우스 | [`tictactoe/`](tictactoe/) |

각 폴더의 `index.html`이 해당 수업의 통합 허브입니다.
리포지토리 루트의 `index.html`은 두 수업을 선택하는 **메인 페이지**입니다.

---

## 사용 방법

- **메인 페이지**: 루트 `index.html`을 열면 수업을 선택할 수 있습니다.
- **수업별 진입**: `crypto/index.html` 또는 `tictactoe/index.html`을 직접 열어도 됩니다.
- 모든 파일은 Self-contained HTML이며 외부 CDN 없이 오프라인으로 작동합니다.

### GitHub Pages 배포

Settings → Pages → Source: `main` / `root` 선택 시,
사이트 루트(`/`)에서 메인 페이지가 열립니다.
