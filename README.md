# 1주차 · 우리들의 암호 이야기

거제교육지원청 영재교육원 **중등수학 2** 수업 자료입니다.

4시간(180분) 블록 수업으로 구성된 암호학 주제의 인터랙티브 학습 자료로,
학생용 웹앱 · 인쇄용 학습지 · 교사용 슬라이드를 포함합니다.

- **대상**: 중학교 2학년 영재교육원 학생
- **수업일**: 2026년 4월 11일
- **담당 강사**: 이동언
- **교육과정 틀**: WEDCR (Warm-up → Explore → Discover → Connect → Reflect)

## 차시 구성

| 차시 | 제목 | 핵심 질문 | 키워드 |
|------|------|---------|--------|
| 1 | 암호의 세계로 | 체크섬은 어떻게 오류를 잡아내는가? | mod 연산, ISBN, Luhn |
| 2 | 비밀을 지켜라 | 좋은 암호에는 어떤 수학이 필요한가? | 시저, 아핀, 서로소, 역원 |
| 3 | 암호를 깨뜨려라 | 통계 패턴으로 암호를 깰 수 있을까? | 빈도 분석, 그래프 슬라이딩 |
| 4 | 소수의 힘 | 소인수분해는 왜 어려운가? | 소수, RSA 공개키 암호 |

4개 차시는 나선형 연결로 설계되어 있습니다:
mod 연산(1차시) → 암호화/복호화(2차시) → 빈도 공격(3차시) → 공개키 해결(4차시)

## 사용 방법

### 오프라인 사용

모든 파일은 Self-contained 단일 HTML입니다. 외부 CDN 없이 작동합니다.

1. 파일을 다운로드합니다.
2. 같은 폴더에 모든 HTML 파일을 둡니다.
3. index.html을 브라우저에서 엽니다.

### GitHub Pages 배포

Settings → Pages → Source: main / root 선택
