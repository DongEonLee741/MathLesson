# 기본 다각형과 틱택토

거제교육지원청 영재교육원 **중등수학 2** 수업 자료입니다.

4차시(180분) 블록 수업으로 구성된 **위상수학** 주제의 인터랙티브 학습 자료로,
학생용 인터랙티브 학습지 · 공간 분석 도구 · 4종 미니게임을 포함합니다.

- **대상**: 중학교 2학년 영재교육원 학생
- **담당 강사**: 이동언
- **교육과정 틀**: WEDCR (Warm-up → Explore → Discover → Connect → Reflect)
- **핵심 개념**: 기본 다각형(Fundamental Polygon), 경계 이어붙이기, 원기둥·토러스·뫼비우스, 위상 불변량

---

## 차시 구성

| 차시 | 제목 | 핵심 질문 | 키워드 |
|------|------|---------|--------|
| 1 | 평면 틱택토의 숨은 수학 | 틱택토는 정말 무승부 게임일까? | 승리 경로, 칸의 가치, 불균등 |
| 2 | 경계를 넘어 — 원기둥·토러스 | 경계를 이으면 게임이 어떻게 달라질까? | 감싸기, 기본 다각형, 동등한 게임 |
| 3 | 뫼비우스 — 뒤집힌 세상 | 반대 방향으로 이으면? | 거울상, 타일링 뷰, 클라인 병 |
| 4 | 위상 게임 축제 | 공간이 바뀌면 게임이 바뀐다 | 당구·오목·미로·색칠, 우주의 모양 |

4개 차시는 나선형으로 연결됩니다:
평면의 수학 구조(1차시) → 원기둥·토러스 탐험(2차시) → 뫼비우스 분석(3차시) → 위상 게임 종합 + 우주 연결(4차시)

## 기술 노트

- **오프라인 동작**: 모든 파일은 self-contained HTML이며, Three.js를 `lib/three.r128.min.js`에 자체 호스팅하므로 인터넷 연결 없이도 작동합니다.
- **저장**: 학습 진행도는 브라우저 `localStorage`에 차시별 prefix(`topo_session{N}_`)로 저장됩니다.
- **차시 간 회수**: 1차시 R-4 예측값이 2차시 W-2에, 3차시 W-1 입력이 3차시 C-1에 자동 회수됩니다.
- **분석기 알고리즘**: ghost-tile 방식으로 평면 8 / 원기둥 12 / 토러스 12 / 뫼비우스 16 경로를 정확히 산출합니다.

## 수학적 배경

이 수업은 게임판의 **경계를 어떻게 이어붙이느냐**에 따라 같은 게임이 완전히 달라진다는 것을 체험합니다.

- **평면**: 경계가 벽. 승리 경로 8개, 칸의 가치 불균등.
- **원기둥**: 좌우를 같은 방향으로 이음. 경로 12개, 모든 칸 가치 균등.
- **토러스**: 좌우 + 상하를 이음. 경로 12개 (3×3에서는 원기둥과 동일).
- **뫼비우스 띠**: 좌우를 반대 방향으로 이음. 경로 16개, 다시 불균등 (2행이 특별).
- **클라인 병**: 좌우 같은 방향 + 상하 반대 방향. 경로 22개 (2열이 특별).

4차시에서는 당구·오목·미로·지도 색칠 등 다른 게임에서도 같은 위상 원리가 작동함을 확인하고,
마지막에 이 수학이 **실제 우주의 모양**을 탐구하는 도구임을 안내합니다.
